Tips for Success
Do's
  • Follow the exact format shown in sample data
  • Read the instructions sheet carefully
  • Use the currency selector for consistent exports
  • Test with a few rows first before bulk import
Don'ts
  • Don't modify the template headers
  • Don't leave required fields empty
  • Don't import files larger than 10MB
  • Don't ignore validation error messages
File Requirements
