## 💡 รายละเอียดขั้นตอนการใช้งาน (Step-by-Step Guide)

### 1. ฝั่ง Admin: การตั้งค่าและการจัดการ Special Package (CRUD)
*   **การเข้าสู่ระบบเมนู:** ไปที่แถบเมนูด้านซ้าย -> เลือกหัวข้อ `Control Panel` -> คลิก `Special Packages`
*   **การเพิ่มแพ็กเกจใหม่ (Create):**
    1.  กดปุ่ม `+ Create Special Package` 
    2.  กรอกรายละเอียดข้อมูลทั่วไป เช่น *ชื่อแพ็กเกจ, รหัสทัวร์, ราคาผู้ใหญ่/เด็ก ต่อหัว, สกุลเงิน, ระยะเวลาวัน*
    3.  ที่หัวข้อ **Day-by-Day Itinerary Planner** กด `Add Day` เพื่อออกแบบบริการรายวัน
    4.  ภายในแต่ละวัน เลือกประเภทกิจกรรมย่อย (Hotels, Transfers, Excursions, Flights, Others) และป้อนราคาที่ตกลงไว้กับ Supplier
    5.  กดปุ่ม `Save Package` เพื่อบันทึกเข้าสู่ระบบ
*   **การแก้ไข (Edit):** กดปุ่ม `Edit` ท้ายแถวของตารางแพ็กเกจทัวร์ เพื่อเข้าไปปรับเปลี่ยนราคา หรือเปลี่ยนแผนการท่องเที่ยวรายวันตามการอัปเดตของ Supplier
*   **การเปิด/ปิดการใช้งาน (Disable/Enable):** กดปุ่ม `Disable` หรือ `Enable` เพื่อซ่อนหรือเปิดสิทธิ์การเข้าถึงแพ็กเกจนี้ให้กับ Agent
*   **การลบ (Delete):** กดปุ่ม `Delete` เพื่อลบข้อมูลทัวร์สำเร็จรูปออกอย่างถาวร

---

### 2. ฝั่ง Agent: การสร้าง Quotation และการคำนวณราคา
*   **การป้อนข้อมูลทั่วไป:** เข้าหน้าสร้างใบเสนอราคา (`add_trip.html`) -> ป้อนชื่อลูกค้า รายละเอียดผู้เดินทาง และจำนวนคนเดินทาง (**Adults / Children**)
*   **การป้อน Trip Start Date:** เลือกวันเริ่มต้นเดินทางให้เรียบร้อยก่อนทำการเลือกแพ็กเกจ
*   **การเลือก Special Package (Optional):**
    *   **หากเลือกใช้แพ็กเกจทัวร์สำเร็จรูป:** เลือกทัวร์ที่ต้องการในหัวข้อดรอปดาวน์ **Special Package (Optional)**
        *   ระบบจะดึงข้อมูลบริการย่อยทั้งหมด (โรงแรม, รถรับส่ง, เที่ยวบิน) มากระจายลงตารางย่อย และคำนวณวันเดินทางเข้าพัก/วันที่ทำกิจกรรมอ้างอิงตาม *Trip Start Date* โดยอัตโนมัติ
        *   ระบบจะล็อกการแก้ไขตารางบริการย่อย (ปุ่มเพิ่ม/แก้ไข/ลบ จะถูกซ่อน) เพื่อป้องกันไม่ให้ Agent เข้าไปแก้ไขความถูกต้องของแพ็กเกจหลัก
        *   ระบบจะคำนวณราคายอดรวมให้อัตโนมัติ: `ราคารวม = (ราคาผู้ใหญ่ต่อหัว × จำนวนผู้ใหญ่) + (ราคาเด็กต่อหัว × จำนวนเด็ก)`
    *   **หากจัดทริปเองแบบอิสระ (Custom):** ให้เลือกเป็น `-- None (Custom Quotation) --` เพื่อให้ Agent สามารถป้อนตารางโรงแรมและกิจกรรมได้เองเหมือนเดิม
*   **การเสนอราคาและการจอง:** กดปุ่ม `Save Quotation` และเมื่อลูกค้าอนุมัติตกลงจอง ให้กดปุ่ม `Convert to Booking` เพื่อแปลงข้อมูลเป็นใบจองเสนอไปยัง Admin

---

### 3. ฝั่ง Admin: การแก้ไขและเปลี่ยนโรงแรมเมื่อห้องพักเต็ม (Booking Stage)
*   **การตรวจสอบรายการจอง:** เข้าสู่เมนู `Control Panel` -> เลือก `Bookings` -> กดแก้ไขใบจองรายการล่าสุดที่มีการแจ้งเตือนเข้ามา (`edit_booking.html`)
*   **การเปลี่ยนโรงแรมทดแทน:**
    1.  ไปที่หัวข้อตาราง **Hotel Details** (รายละเอียดโรงแรม)
    2.  กดปุ่ม **View (รูปสมุดคู่มือสีเทา)** ท้ายแถวโรงแรมที่พักเต็ม
    3.  หน้าต่างแก้ไขรายการโรงแรมจะปรากฏขึ้นมา ให้เลือก **City (เมือง)** หรือ **Hotel (โรงแรม)** ตัวใหม่ที่ต้องการเปลี่ยนทดแทน พร้อมทั้งปรับปรุงประเภทห้องพักหรือราคา
    4.  กดปุ่ม **Save Hotel** เพื่ออัปเดตรายละเอียดโรงแรมใหม่ลงสู่ตารางหลักชั่วคราว
*   **การบันทึกใบจองหลัก:** เมื่อทำการเปลี่ยนโรงแรมเรียบร้อยแล้ว ให้กดปุ่ม **Save Booking (บันทึกใบจอง)** ปุ่มสีเขียวที่ด้านล่างสุดของฟอร์มหลัก เพื่ออัปเดตการเปลี่ยนแปลงทั้งหมดเข้าสู่ฐานข้อมูลและส่งแจ้งเตือนกลับไปยังหน้าจอของ Agent
